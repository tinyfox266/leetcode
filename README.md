# 注意事项
* For string ,"==" and "equals" have different meaning
    * "==" compare reference
    * "equals" compare value
    
# 边界条件
* array
    * array's length is 0
* tree
    * root is null   
    * leaf
* list 
    * head is null
* int
    * the result is more than Integer.MAX_VALUE 
    * the result is less than Integer.MIN_VALUE

* The Boyer-Moore Majority Vote Algorithm 
    * 找出一个数组中出现次数超过n/2的元素
    * O(n) 时间复杂度
    * O(1) 空间复杂度
