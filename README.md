# 注意事项
* For string ,"==" and "equals" have different meaning
    * "==" compare reference
    * "equals" compare value
    
# 边界条件
* array
    * array's length is 0
* tree
    * root is null   
    * leave
* list 
    * head is null
* int
    * the result is more than Integer.MAX_VALUE 
    * the result is less than Integer.MIN_VALUE
